# Institutional Equity Market Regime Research Platform

A quantitative research platform built in **R** to model market regime dynamics, tail-risk escalation, and crisis belief evolution.

---

## Project Overview

This project studies market transitions across latent states:

- Calm
- Stress
- Crisis

It combines:

- empirical return diagnostics
- Gaussian deviation testing
- bootstrap uncertainty estimation
- latent regime switching
- crisis shock surface modeling
- belief velocity dynamics
